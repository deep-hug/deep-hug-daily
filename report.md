## 项目说明

### 目录结构

```
├── assets         // 主题、字体、全局样式等静态资源
├── components             // 全局公用组件
├── layout              // 菜单组件
│   ├── components         // 菜单封装
│   ├── index.vue         // 菜单出口
├── mock               // mock数据结构
├── views                  // 网站页面模块
├── requert                    // 所有请求
├── router                 // 路由
│   ├──  asyncRoutes              // 路由新增模块
│   ├──  index              // 路由配置文件
│   ├──  staticRoutes             // 通用路由
├── store                  // 全局 store管理
├── utils                  // 全局公用方法
├── App.vue                // 入口页面
├── index.html                 // html模板
├── main.js              // 入口 加载组件 初始化等
└── package.json               // package.json

```
