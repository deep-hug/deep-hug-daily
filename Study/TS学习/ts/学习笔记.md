### 如何检验 ts 安装成功

```js
    tsc -v // 能够出来版本号就代表安装成功
```

### ts 开发工具 vxcode 自动编译 .ts 文件

```js
    tsc --init // 会生成一个 tsconfig.json 的文件，修改其文件的 outDir: './js' ，然后按顺序点击 终端 -> 运行任务 -> typescript -> tsc: 监视。就可以自动编译 .ts 文件了，不用每次再执行 tsc index.ts 这样的命令行了。
```
